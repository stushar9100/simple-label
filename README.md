### Google Label Generator Add On Creation


## Init Prompt
The task is to create a label generation add on for google apps. Similar to https://foxylabels.com
User flow is as follows:
User opens google sheet and enters like a Shipping information in columns and rows. Example: Column A: Name
Column B: Address
Column C: Phone

User then clicks on the Add On which shows a Modal to include the columns in the label.

User can format the label structure and then click on a Create button, which then transforms the rows in the sheet into a New Google Doc in a printable format like rectangular blocks for each info.
